# Collange

**How to clean:**

./gradlew clean

**How to build:**

./gradlew build

**How to test:**

./gradlew test

**How to clean, build, test:**

./gradlew clean build test

**How to dry-run deployment:**

./gradlew clean stage

**How to dry-run deployment, no unit test execution:**

./gradlew clean stage -xtest
